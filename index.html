NixOS Binary Cache Cache for 35c3

We are making your installations faster without compromising security as all packages are officially signed and only transparently forwarded.

CLI Usage

# in the installation iso
nixos-install --option substituters https://cache.nsupdate.info
# After rebooting and trying out new config
nixos-rebuild switch --option substituters https://cache.nsupdate.info
# When building a single package (only as root or nix.trustedUser)
nix build -f '' hello \
  --option substituters https://cache.nsupdate.info

NixOS Config

This will persist the binary cache config to be used automatically at run-time:
In your configuration.nix 

nix.binaryCaches = [ "https://cache.nsupdate.info" "https://cache.nixos.org/" ];
